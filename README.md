# 📊 CodeAlpha - Unemployment Analysis with Python

## 📌 Project Overview

This project is developed as part of the **CodeAlpha Machine Learning Internship**.

The objective of this project is to analyze unemployment trends in India using Python. The analysis focuses on cleaning and exploring the dataset, visualizing unemployment patterns across different states, and studying the impact of the COVID-19 pandemic on unemployment rates. The project also identifies key trends and provides insights that can help understand the country's employment situation.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing.
- Explore unemployment data using Python.
- Visualize unemployment trends across India.
- Analyze the impact of COVID-19 on unemployment.
- Identify state-wise unemployment patterns.
- Generate insights that could support economic and social policy decisions.

---

## 📂 Dataset

**Dataset Name:** Unemployment in India

**Source:** Kaggle

https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

The dataset contains information such as:

- Region (State)
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

## 🦠 COVID-19 Impact Analysis

One of the primary goals of this project is to study how the COVID-19 pandemic affected unemployment in India.

The analysis highlights:

- Sharp increase in unemployment during lockdown.
- Changes in labour participation rates.
- State-wise differences in unemployment levels.
- Employment recovery trends after lockdown.

---

## 📌 Key Findings

- Unemployment rates increased significantly during the COVID-19 lockdown.
- Different states experienced varying levels of unemployment.
- Labour participation rates fluctuated during the pandemic.
- Employment gradually recovered after the lockdown period.
- Data visualization makes it easier to understand long-term unemployment trends.
